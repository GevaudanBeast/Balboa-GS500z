## 🌍 Contributing Translations / Contribuer aux traductions

We welcome translations to other languages!
Nous accueillons les traductions dans d'autres langues !

To contribute a translation:
1. Copy an existing `.md` file
2. Translate the content
3. Save with the appropriate language suffix (e.g., `.de.md` for German)
4. Submit a pull request

Pour contribuer une traduction :
1. Copiez un fichier `.md` existant
2. Traduisez le contenu
3. Enregistrez avec le suffixe de langue approprié (ex : `.de.md` pour l'allemand)
4. Soumettez une pull request
